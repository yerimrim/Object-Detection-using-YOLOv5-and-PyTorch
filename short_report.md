# Short Report

I used an image containing multiple everyday objects as the input for object detection. 
The YOLOv5 models detected several objects, including people and other items present in the image. 
Overall, the YOLOv5x model provided more accurate detection results compared to YOLOv5s. 
I modified the original code by changing the YOLO model from YOLOv5s to YOLOv5x, printing the detected object names with confidence scores, and saving the output image with a custom file name. 
The model correctly identified most objects in the image, although a few confidence scores varied between models. 
README.md is important because it explains how to use and understand the project. 
The requirements.txt file is essential because it allows other developers to install the exact dependencies needed to reproduce the project.
